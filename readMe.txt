// This is simple application for weather information by Rajkumar Sangram Surnar

Step 1: Download Application. and open terminal & change your current directory to application
Step 2: Run npm install in terminal
Step 3: Run application as node server.js in terminal & open localhost:3000 in browser

// How to use
//Part I
Step 1: Select one or more cities on left side options & click on Show Weather button. It will display weather information of 14 days for        
        selected cities.

//Part II
Step 1: Click on My City Weather(GPS Based) button. It will display weather information of for your current location city.
